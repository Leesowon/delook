## 답변

클로저는 함수가 선언될 당시의 외부 렉시컬 환경을 기억하고 접근할 수 있게 해주는 함수입니다.  
즉, 외부 함수의 변수에 접근할 수 있는 내부 함수라고 이해하면 됩니다.  
반면, 스코프는 변수의 유효 범위를 의미하며, 클로저가 존재할 수 있는 기반이 되는 개념입니다.  
클로저는 주로 상태 유지, 은닉화 text 등에 사용되고, 자바스크립트의 비동기나 이벤트 핸들링에서 자주 등장합니다.

#### 예시

```js
function counter() {
  let count = 0;
  return function () {
    count += 1;
    return count;
  };
}

const increase = counter();
console.log(increase()); // 1
console.log(increase()); // 2
```
